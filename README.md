📌 專案介紹

台中輕晴 是一個以「北歐霧面 × 軟霧藍色調」為主視覺的天氣介面設計。
目標是打造一個 打開就覺得舒服、資訊清楚、帶有溫度感的小型天氣工具。

介面靈感來自：
	•	台中的晴朗城市氣氛
	•	毛玻璃半透明的柔和質感
	•	心理學中的「溫柔提醒」概念

使用者打開頁面後，可以快速掌握：
	•	當前天氣狀態
	•	平均體感溫度
	•	天氣描述
	•	降雨建議 & 穿搭建議
	•	稍後時段的逐時天氣

並搭配一句 「今日輕晴小語」，讓天氣不只是數據，也能帶來一點情緒上的陪伴。

⸻

🎨 設計理念（簡短版）

這次的介面我希望做得 乾淨、清爽、有點療癒感。
因此使用霧藍色系、毛玻璃卡片和圓角設計，讓資訊看起來不壓迫。
小語的加入則讓整體多了一點「溫度」，不只是查天氣，也像是被安安穩穩地陪著。

📡 功能特色

✔ 即時天氣顯示

顯示台中目前的天氣、平均溫度、天氣圖示。

✔ 穿搭 & 降雨建議

依降雨機率與最高溫給出小提醒（帶傘 or 穿外套）。

✔ 今日輕晴小語

目前為 隨機小語，可升級為「依天氣變換的小語」。

✔ 稍後天氣預報（逐時）

橫向滑動卡片顯示：早晨、中午、下午、晚上、明天等。

⸻

🔗 API 格式（Zeabur 後端）

後端回傳資料格式如下：

{
  "success": true,
  "data": {
    "forecasts": [
      {
        "startTime": "2025-11-29 12:00",
        "weather": "多雲時晴",
        "maxTemp": "26",
        "minTemp": "20",
        "rain": "10%"
      }
    ]
  }
}
